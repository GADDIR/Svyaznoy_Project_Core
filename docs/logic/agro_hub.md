# УЗЕЛ 1.1.1 — АГРО-ПОСТ МАРШРУТА (ПОЛНАЯ СТРУКТУРА)

## 1. ИНФРАСТРУКТУРА И ОБЪЕМ
- **Объект:** Малая («подарочная») палатка гражданского типа.
- **Логика выбора:** Минимальный вес (переноска) + минимальный силуэт (скрытность).
- **Разумение АН:** Страховой полис для возврата в оперативный цикл.

## 2. КОМПЛЕКТАЦИЯ СКЛАДА
Внутри «подарка» строго регламентированный набор:
- **Инструмент:** Лопатка/Мотыга (стационарно).
- **Генетический фонд:** Стак семян (по 5-10 шт каждого вида).
- **Провизия:** 1-2 сушеных овоща (Сухой паек связиста).
- **Вода:** 1 фляга/бутылка (стерилизованная).

## 3. СТРАТЕГИЧЕСКОЕ РАЗУМЕНИЕ
- **Принцип «Транзита»:** Запрет на длительное проживание. Только пополнение, замена и экстренное ХП.
- **Маскировка:** Вживление в «зеленку». Видимость с тропы = Тактическая ошибка.
- **Обновление:** При изъятии ресурса статус узла меняется на `Refill Needed`.

## 4. ТЕХНИЧЕСКИЙ РЕГЛАМЕНТ (AI)
- **Запрет:** Использование бочек (из-за веса и демаскировки).
- **Состояние:** Палатка всегда находится в статусе `Closed`.
- **Логика:** Статичный объект с динамическим наполнением.
