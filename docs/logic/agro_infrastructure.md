# ЭКОНОМИЧЕСКАЯ ИНФРАСТРУКТУРА СЕКТОРА 900

## 1. ТИПОЛОГИЯ AGRO-HUB (ФЕРМЕРСТВО)
- **Малый объем («Подарочная»):** 1–2 грядки, максимальная скрытность, быстрый цикл.
- **Средний объем (Гражданская):** Точка восстановления вне Сейф-зоны. Место отдыха.
- **Большой объем (Военная):** Стационарный цех под маскировочными сетями.

## 2. МЕНЕДЖМЕНТ ОБОРУДОВАНИЯ (STORAGE)
### Палатка vs Ящик
- **Внешний мир:** Категорический запрет на деревянные ящики. Использование только палаток.
- **Разумение:** Палатку можно свернуть и эвакуировать по «Протоколу Жертвы». Ящик — статичная цель и потеря ресурсов.
- **Сейф-зона (Приют):** Разрешено использование сейфов и ящиков для хранения капитала (деньги, ключи).

## 3. МОДУЛЬ «УМНАЯ БОЧКА» (HYDRO-ENGINEERING)
- **Сбор воды:** При уходе на дистанцию крышка бочки переводится в статус `Open` для сбора осадков.
- **Универсальность:** Использование бочки как очага для копчения «Валюты для Quinn».
- **Гигиена:** Обязательное кипячение (`BoilWater`) всей собранной воды перед употреблением.

## 4. МЕДИЦИНСКИЙ ФАКТОР СЕЙФ-ЗОНЫ
- **Константа:** Регенерация ХП в Сейф-зоне заблокирована.
- **Логика АН:** Сейф-зона — только для логистики и торгов. Для лечения Некрасов обязан переместиться в Фермерский лагерь или Лагерь отдыха в лесу.

## 5. ДИРЕКТИВА ШТАБА «В»
1. **Приоритет эвакуации:** При угрозе первым сворачивается палатка (вместилище), а не отдельные предметы.
2. **Агро-контроль:** Проверка состояния бочки (сбор воды) перед началом посадочного цикла.
