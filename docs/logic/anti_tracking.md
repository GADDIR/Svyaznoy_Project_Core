# Протокол: КОНТРРАЗВЕДКА И «РАЗРЫВ ШАБЛОНА»

## 1. АЛГОРИТМЫ СКРЫТНОГО ДОСТУПА
- **Anti-Tracking (Петли):** Запрет прямого маршрута к узлам 1.2 и 1.3. Обязательное посещение "пустых" локаций перед входом в зону склада.
- **Randomization:** Выбор точки хранения через генератор случайных чисел из списка доступных Архивов.
- **Silent Hour:** Приоритетный доступ к Deep Cache в условиях тумана, ливня или ночи.

## 2. ДЕЗИНФОРМАЦИЯ (БАЛЛАСТНЫЕ УЗЛЫ)
- **Объект:** Малая палатка на видном месте.
- **Содержимое:** Минимальный НЗ-1 (использованный бинт, кости, пустая банка).
- **Цель:** Насыщение любопытства врага "пустой" добычей для защиты основного схрона.

## 3. СЕНСОРНЫЙ БАРЬЕР
- **Триггер:** Обнаружение игрока в радиусе 500м.
- **Действие:** Полная блокировка посещения складов. Переход в паттерн "Обычный выживший".
