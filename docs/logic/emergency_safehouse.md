# МОДУЛЬ «ТОЧКА ЗЕРО» (EMERGENCY SAFEHOUSE)

## 1. СТРАТЕГИЧЕСКИЙ ТРИГГЕР
Активация узла происходит при выполнении условий:
- `Base_Compromised == TRUE` (Захват/Обнаружение основного лагеря).
- `Equipment_Status == CRITICAL` (Потеря или полное разрушение снаряжения).

## 2. СОСТАВ «НАБОРА НЕКРАСОВА» (CORE KIT)
Неприкосновенный запас (НЗ) для возврата в оперативный цикл:
- **Инструмент:** Дубликат ножа (Сталь/Кость) для запуска «Костяного цикла».
- **Огонь:** Стак коры (8 шт.) + Hand Drill Kit.
- **Медицина:** Пакет собственной крови + 1 стерильный бинт (База «К.У.Л.А.К.»).
- **Данные:** Координаты резервных схронов.

## 3. ЛОГИКА «РАЗУМЕНИЯ»
- **Скрытность 200%:** Запрет на наземные объекты. Только подземные схроны или дупла.
- **Дисциплина:** Категорический запрет на использование ресурсов узла для бытовых нужд или торговли с Quinn.
- **Регенерация узла:** После вскрытия НЗ, АН обязан восполнить запас в течение 24 игровых часов.

## 4. ПОВЕДЕНЧЕСКИЙ ПАТТЕРН
При потере снаряжения АН переходит в режим «Тень»: скрытное движение к Точке Зеро вне дорожной сети Сектора 900.
