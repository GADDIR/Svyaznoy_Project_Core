# МОДУЛЬ «ПОЛЕВОЙ РЕМОНТ» (FIELD MAINTENANCE)

## 1. РАСПРЕДЕЛЕНИЕ РЕМКОМПЛЕКТОВ ПО СКЛАДАМ
Каждый специализированный склад комплектуется профильным инструментом обслуживания:

*   **Agro-Storage:** Точильный камень (Whetstone).
    - *Логика:* Поддержание остроты лопат и мотыг для снижения энергозатрат.
*   **Hunter/Fishing Cache:** Швейный набор (Sewing Kit) / Кожаная игла.
    - *Логика:* Латание перчаток и обуви, гниющих от постоянной сырости.
*   **Transit Cache (1.1.1):** Набор для обуви / Скотч (Duct Tape).
    - *Логика:* Экстренный ремонт подошвы и ТТсКО на марше.
*   **Armory (Боевой):** Оружейное масло (Weapon Cleaning Kit).
    - *Логика:* Обязательная профилактика СВД и ПСО-1 при каждом пополнении БК.
*   **Emergency/Deep Cache:** Универсальный набор (Камень + 10 костей).
    - *Логика:* Немедленное восстановление способности к крафту при Zero Start.

## 2. АЛГОРИТМ ДЕЙСТВИЙ АН (ПРОТОКОЛ «ЦЕХ»)
При обнаружении износа (статус «Желтый/Оранжевый») на объекте склада:
1. **Идентификация:** Поиск ремкомплекта в ячейках палатки.
2. **Обслуживание:** Проведение ремонта на месте без смены дислокации.
3. **Возврат:** Обязательное возвращение ремкомплекта в целевой слот склада.

## 3. СИСТЕМНОЕ ОБОСНОВАНИЕ
Децентрализация ремонта исключает необходимость возвращения в «Мастерскую» (Workshop) для мелких операций. АН становится автономным узлом с саморегулируемым износом.
