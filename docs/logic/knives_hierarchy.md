# Протокол: Технологическая Иерархия «Острие Выживания» (Project: NEKRASOV)

## 1. Общие положения
Данный документ описывает когнитивную модель поведения АН при работе с режущими инструментами. 
Цель: Обеспечение 100% автономности в условиях карты Banov.

## 2. Спецификация инструментов

### [A] Каменный нож (Stone Knife)
*   **Тип:** Расходник «нулевого цикла».
*   **Приоритет:** Минимальный.
*   **Логика:** Сделал -> Добыл кость -> Выбросил.
*   **Запрет:** ИИ запрещено хранить более 1 единицы или пытаться чинить.

### [B] Костяной нож (Bone Knife)
*   **Тип:** Основной операционный инструмент.
*   **Приоритет:** КРИТИЧЕСКИЙ.
*   **Логика:** Поддержание стака (10 ед. сырья). Регулярный цикл замены.
*   **Назначение:** Разделка, крафт, быт.

## 3. Матрица принятия решений (Logic Gate)
- IF (MainKnife.Condition == "Orange") -> START (GatherStones)
- IF (BonesInInventory < 10) -> START (Hunt/ScavengeBones)
- IF (BoneKnife.Exists == TRUE) -> LOCK (SteelKnife for Utility)

## 4. Инструкция для Штаба «В»
Блокировать любые действия ША (ИИ-ассистента), ведущие к нецелевому расходу прочности стального клинка при наличии костяного аналога.
