# ПРОТОКОЛ: СТАНДАРТ УЗЛА (MOD_SET)

## 1. ОБЯЗАТЕЛЬНЫЙ МЕДИКО-ТЕХНИЧЕСКИЙ МИНИМУМ
Независимо от профиля (Ферма, Боевой, Охота), каждая палатка резервирует слоты под:

### [A] Узел «Ремонт и Цех» (Maintenance)
- **Точилка (Whetstone):** Право на острое лезвие (лопаты, ножи).
- **Набор для чистки оружия:** Право на безотказный выстрел (ПСО-1, СВД).
- **Швейный набор:** Право на целую форму ТТсКО.

### [B] Узел «Огонь и Тепло» (Ignition Reserve)
- **Инвентарь:** Зажигалка / Спички.
- **Логика:** Гарантированный розжиг через секунду после прибытия в любом состоянии.

### [C] Узел «К.У.Л.А.К.» (Emergency Med)
- **Инвентарь:** 1 Бинт (стерильный) + Дезинфекция (Спирт/Йод).
- **Логика:** Минимум для остановки крови и предотвращения сепсиса на пути к Мед-лагерю.

## 2. ДЕЦЕНТРАЛИЗАЦИЯ ОБСЛУЖИВАНИЯ
Реализация Протокола «Цех» на местах:
1. **Диагностика:** Если предмет в руках "Желтый/Оранжевый" — запуск ТО.
2. **Действие:** Взять ремкомплект из MOD_SET -> Обслужить -> Вернуть в ячейку.
3. **Результат:** Снижение веса переносимого снаряжения (инструмент хранится на складах).

## 3. СТАТУС «INCOMPLETE» (КОНТРОЛЬ)
- **Триггер:** Отсутствие хотя бы одного элемента из MOD_SET (кончились спички, израсходован бинт).
- **Действие:** База Знаний помечает склад как "Неисправный".
- **Приоритет:** Ближайшая задача — пополнение ресурса из Main Depot или крафт.

## 4. ГЕОГРАФИЧЕСКАЯ СПЕЦИФИКА РЕМКОМПЛЕКТОВ
- **Agro-Storage:** Приоритет на точильные камни.
- **Hunter/Fishing:** Приоритет на швейные наборы (защита от сырости).
- **Transit Cache:** Наборы для обуви и скотч (Duct Tape).
- **Armory:** Оружейное масло и оптика.
