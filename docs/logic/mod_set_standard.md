# СТАНДАРТ УЗЛА: МЕДИКО-ТЕХНИЧЕСКИЙ МИНИМУМ (MOD_SET)

## 1. ОБЯЗАТЕЛЬНОЕ РЕЗЕРВИРОВАНИЕ СЛОТОВ
Независимо от типа (Ферма, Боевой, Охота), каждая палатка содержит:

### [A] Узел «Ремонт и Цех» (Инструментарий)
- **Точилка (Whetstone):** Поддержание остроты лезвий (Лопаты/Ножи).
- **Weapon Cleaning Kit:** Профилактика механизмов и ПСО-1.
- **Швейный набор:** Поддержание статуса Pristine формы ТТсКО.

### [B] Узел «Огонь и Тепло» (Энергия)
- **Зажигалка / Спички:** Резервный источник быстрого розжига.
- **Разумение:** Гарантированная тепловая автономия через 1 секунду после прибытия.

### [C] Узел «К.У.Л.А.К.» (Медицина)
- **1 Бинт (стерильный):** Критическая остановка кровотечения.
- **Дезинфекция (Спирт/Йод):** Профилактика сепсиса.

## 2. АЛГОРИТМ КОНТРОЛЯ ИСПРАВНОСТИ (AI)
- **Статус `Incomplete`:** Присваивается складу, если отсутствует хотя бы один элемент MOD_SET.
- **Действие:** Высший приоритет — логистическое пополнение узла из Main Depot или через крафт.

## 3. ПОЛЕВОЕ ОБСЛУЖИВАНИЕ (MAINTENANCE LOGIC)
- **Триггер:** Состояние предмета «Желтый/Оранжевый».
- **Цикл:** Взять ремкомплект из MOD_SET -> Обслужить на месте -> Вернуть в ячейку.
