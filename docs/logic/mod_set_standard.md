# ПРОТОКОЛ: СТАНДАРТ УЗЛА (МОДУЛЬ НЗ-1)

## 1. СОСТАВ КОМПЛЕКТАЦИИ «НЗ-1» (УТВЕРЖДЕНО)
В каждой палатке сети «Приют» резервируется фиксированный блок слотов под неприкосновенный запас:

### [A] Тех-блок (Maintenance)
- **Точилка (Whetstone):** Обслуживание холодного оружия и инструмента (лопаты, мотыги).
- **Набор для чистки оружия:** Профилактика огнестрельного парка (ПСО-1, СВД).
- **Швейный набор:** Поддержание целостности формы ТТсКО.

### [B] Огневой блок (Energy)
- **Зажигалка / Спички:** Гарантия терморегуляции и стерилизации.

### [C] Мед-блок (Medical)
- **1 Бинт (стерильный):** Остановка критического кровотечения.
- **Дезинфекция (Спирт/Йод):** Санитарная обработка ран (К.У.Л.А.К.).

## 2. АЛГОРИТМ РАЗУМЕНИЯ АН
- **Приоритет обслуживания:** НЗ-1 — первый объект проверки при посещении склада.
- **Дисциплина расхода:** Ресурсы НЗ-1 используются строго на месте по принципу «Взял — Обслужил — Вернул».
- **Запрет на вынос:** Инструментарий НЗ-1 запрещено забирать в рейд (исключение: «План Б»).
- **Статус «Пусто»:** При расходе любого компонента (бинт, масло) База Знаний ставит задачу на обязательное восполнение при следующем визите.

## 3. ВИЗУАЛЬНАЯ ГИГИЕНА (ШТАБ «В»)
Предметы НЗ-1 должны группироваться в **нижнем ряду ячеек** инвентаря палатки, формируя узнаваемый паттерн «Аптечка-Мастерская» для мгновенного считывания статуса готовности узла.

## 4. ДЕЦЕНТРАЛИЗАЦИЯ
Логика "Цех" теперь привязана к точкам хранения. Снижение веса переносимого снаряжения достигается за счет хранения тяжелого рем-инструмента в модулях НЗ-1 на местах.
