# Протокол: Ресурсная гибкость и Огонь

## 1. Текстильная растопка (Rags as Kindling)
Тряпки — универсальный связующий элемент между медициной и теплом.

### Логика получения:
- Источник: Одежда Ruined или не соответствующая Манифесту ТТсКО.
- Инструмент: Костяной нож (для экономии стали).

### Иерархия инициации пламени:
1. Кора (Экономия ресурсов).
2. Бумага.
3. Грязные/Окровавленные тряпки (Гигиеническая утилизация).
4. Бинт (КРИТИЧЕСКИЙ ЗАПРЕТ на сжигание).

## 2. Итоговая формула костра (Kindling + Fuel)
АН Некрасов обязан иметь минимум 2 способа розжига:
- Природный (Кора + Палки).
- Техногенный (Тряпки + Дрова).
