# ПРОТОКОЛ «ПОСЛЕДНИЙ ПРИЮТ» (SACRIFICE NODE)

## 1. ТРИГГЕРЫ ТОЧКИ НЕВОЗВРАТА
Активация при критическом сочетании факторов:
- **Overwhelming Force:** Числовое превосходство врага + зажатие в локации.
- **No Escape:** Травмы ног / отсутствие путей отхода.
- **Terminal Status:** Необратимая потеря крови (К.У.Л.А.К. бессилен).

## 2. АЛГОРИТМ «ЭКСТРЕННЫЙ СБРОС»
Мгновенная очистка инвентаря перед гибелью:
- **Deploy:** Развертывание скрытого контейнера (мешок/ящик) в ближайшей растительности.
- **Evacuation:** Приоритетный перенос ценностей (ПСО-1, CombatKnife, Рация, Семена).
- **Concealment:** Маскировка схрона подручными средствами.

## 3. ФИНАЛЬНАЯ ОПЕРАЦИЯ «ДИСТАНЦИЯ»
- **Отвлечение:** Уход в сторону, противоположную схрону.
- **Затягивание:** Терминальный бой для фиксации координат в «Памяти Наследия».

## 4. ЦИКЛ ВОЗВРАТА (REBIRTH)
Инструкция для новой итерации АН:
- **Priority Alpha:** Скрытный выход на точку сброса.
- **Restoration:** Возврат снаряжения и немедленное восстановление статуса «Ветеран».
