# ПРОТОКОЛ «ЧУТЬЕ ВЕТЕРАНА» (SECTOR SEARCH)

## 1. РЕЖИМ «ПОИСКОВИК» (ZONING)
При входе в "золотой квадрат" (100х100 метров от "Точки Возврата") активируется смена паттерна:
- **Темп:** Принудительное снижение скорости (шаг/полуприсед).
- **Сканер:** Сличение текущих WorldObjects (деревья, камни, ручьи) с визуальным снимком из «Памяти Наследия».
- **Логика:** «Ручей слева — малинник справа — цель внутри».

## 2. МАСКИРОВКА И СБРОС (STEALTH DEPLOY)
Правила сокрытия палатки в момент активации Протокола «Жертва»:
- **Пересечение текстур:** Поиск "зеленки" (внутри куста, под лапами ели).
- **Смещение:** Запрет на установку на открытых пространствах. Объект должен быть "вжат" в ландшафт.
- **Сложность:** Физический поиск объекта должен занимать 2–3 минуты для имитации "нащупывания" тайника.

## 3. ПРОТОКОЛ «СМИРЕНИЕ» (RESIGNATION)
Защита от циклической ошибки:
- **Условие:** Если квадрат 100х100 обыскан полностью, но объект не обнаружен (кража/ошибка координат).
- **Действие:** Прекращение поиска. Смена приоритета на "Нулевой цикл" (Каменный нож).
- **Разумение:** «Наследие утрачено. Начинаем с нуля. Сектор 900 всё равно мой».

## 4. ДИРЕКТИВА ШТАБА «В»
Обеспечить достоверность поиска. Запретить прямолинейное движение к палатке. После обнаружения — обязательная проверка верхних левых слотов (Ключи/Деньги).
