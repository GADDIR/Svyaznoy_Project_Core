# РЕЕСТР ФУНКЦИОНАЛЬНЫХ УЗЛОВ (СЕТЬ «ПРИЮТ»)

## 1. ЭКОНОМИЧЕСКИЙ КЛАСТЕР
- **Agro-Hub (Фермерский):** 3 грядки, запас воды, склад семян. Минимум демаскировки.
- **Main Depot (Склад):** Основное хранилище под маскировочными сетями. Ресурс для Quinn.
- **Deep Cache (Резерв):** Подводные/подземные схроны. Дубликат на случай захвата.

## 2. ТЕХНИЧЕСКИЙ И ЛОГИСТИЧЕСКИЙ КЛАСТЕР
- **The Workshop (Мастерская):** Бочка/печь. Центр протокола «Цех» (заточка, шитье, ПСО-1).
- **Transit Point (Транзит):** Перевалочный пункт. Запас сухих пайков и воды.
- **Bivouac (Бивуак):** Навес на одну ночь. План «Б».

## 3. БОЕВОЙ И ЭКСТРЕННЫЙ КЛАСТЕР
- **Forward Post (Пост):** Скрытая точка обзора. БК и медицина «К.У.Л.А.К.».
- **Emergency Safehouse (Сейфхаус):** Критический набор (нож, спички, кровь, одежда).
- **Med-Station (Медпункт):** Стерильная зона. Дезинфекция и хирургия.

## 4. ГРАЖДАНСКИЙ КЛАСТЕР
- **Rest Zone (Отдых):** Коптильня, сушка овощей, психологическая разгрузка.

## 5. ЛОГИКА РАЗУМЕНИЯ (AI PRIORITY)
*   **Специализация:** Запрет на копчение мяса в боевом лагере. Запрет на хранение БК в фермерском.
*   **Маскировка:** Боевые/Экстренные узлы — полная невидимость. Технические — допустима маскировка объектов.
