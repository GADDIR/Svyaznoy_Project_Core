# Протокол: НЕКРАСОВ-ПРОВОДНИК (STALKER LOGIC)

## 1. УСЛУГА «ПУТЬ СВЯЗИСТА»
- **Суть:** Проводка сторонних игроков по безопасным маршрутам Сектора 900.
- **Безопасность:** Использование "Гостевых троп". Категорический запрет на демонстрацию Архивов 1.3 и Main Depot 1.2.
- **Контроль:** Если ведомый сокращает дистанцию или достает оружие без команды — немедленный переход АН в боевой режим.

## 2. КОММЕРЧЕСКАЯ ЛОГИКА (БАРТЕР)
- **Фактор Доверия:** Группы (Gost/Bizon) — приоритет 100% (бесплатно). Чужаки — оплата ресурсом.
- **Приоритет оплаты:** Диктуется дефицитом в Базе Знаний (Патроны -> Еда -> Инструмент).
- **Отказ:** При агрессивном поведении клиента или обнаружении "хвоста".

## 3. ТРАНЗИТНЫЙ СЕРВИС
- **Остановки:** Использование Узлов 1.1.1 и 4.1 для отдыха клиента.
- **Доп. услуги:** Монетизация ресурсов (продажа сушеных овощей, использование НЗ-1 для перевязки клиента).

## 4. ТЕХНИЧЕСКИЙ НАВЫК «СОПРОВОЖДЕНИЕ»
- **Дистанция:** Динамический контроль удаления ведомого. Ожидание при отставании.
- **Сенсорика:** Постоянный LookAround360 для защиты "товара" (клиента) и себя.
