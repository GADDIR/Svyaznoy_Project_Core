# Протокол: СЕРВИСНЫЙ ХАБ И ЛОГИКА ПРОВОДНИКА

## 1. УСЛУГА «ПУТЬ СВЯЗИСТА» (STALKER LOGIC)
- **Суть:** Проводка игроков по безопасным "Гостевым тропам" в обход Архивов 1.3 и Main Depot 1.2.
- **Динамика:** Контроль дистанции. При нарушении зоны безопасности или извлечении оружия клиентом — немедленный боевой режим.
- **Сенсорика:** Постоянный LookAround360 для защиты группы.

## 2. МАСТЕРСКАЯ КАК ТЕХНО-ХАБ (SERVICE LOGIC)
- **Визуальное доминирование:** Демонстрация быта (чистка картофеля, работа с ПСО-1) для укрепления авторитета.
- **Информационная гигиена:** "Золотой фонд" (ключи, деньги) переносится в Deep Cache до прибытия клиента.
- **Ограничение доступа:** Клиент видит только "внешнее" оборудование (бочка, пень-верстак). Внутренний инвентарь палатки заблокирован.

## 3. БЕЗОПАСНОСТЬ НА ОБЪЕКТЕ
- **Зона ожидания:** Клиент фиксируется у очага. Любое движение за спину АН — прерывание работ.
- **Слепая навигация:** Вход/выход через "петли" и густую растительность для сокрытия координат мастерской.

## 4. ЭКОНОМИКА И БАРТЕР
- **Фактор Доверия:** Дружественные группы (Gost/Bizon) — приоритет 100%. Чужаки — оплата по дефициту (Патроны -> Еда -> Инструмент).
- **Аренда НЗ-1:** Платное использование инструментов Некрасова клиентом на месте.
- **Бартер трудом:** "Проводка за работу" (полив грядок в Agro-Hub, заготовка дров).

## 5. ТРАНЗИТНЫЙ СЕРВИС
- **Пит-стоп:** Использование узлов 1.1.1 и 4.1 для отдыха и монетизации ресурсов (продажа сухпайков в пути).
