# РЕЕСТР СПЕЦИАЛИЗИРОВАННЫХ СКЛАДОВ

## 1. ТИПОЛОГИЯ И СОСТАВ
### [A] Фермерский склад (Agro-Storage)
- **Состав:** Лопаты, мотыги, лейки, избыток семян.
- **Логика:** Привязка к Узлу 1.1. Запрет на хранение БК (риск коррозии).

### [B] Охотничий склад (Hunter’s Cache)
- **Состав:** Капканы, ножи, шкуры, соль, патроны малого калибра (.22 LR).
- **Локация:** Глухой лес, пути миграции зверя.

### [C] Рыболовный склад (Fishing Point)
- **Состав:** Удочки, костяные крючки, веревки, черви.
- **Локация:** Скрытая береговая линия (камыш/скалы).

### [D] Маршрутный склад (Transit Cache)
- **Состав:** Обувь, сухой паек, фляга, 1 бинт. Приоритет — расходники.
- **Локация:** Точки пересечения (Узел 1.1.1).

### [E] Боевой склад (Armory)
- **Состав:** БК, чистка оружия, глушители, оптика.
- **Разумение:** Стерильность и сухость. Точка с обзором сектора.

### [F] Резервный склад (Deep Cache / Archive)
- **Состав:** Дубликаты ключей, ПСО-1, валюта.
- **Локация:** Сейф-зона (защита) или под водой (инкогнито).

## 2. АЛГОРИТМ РАЗУМЕНИЯ (AI)
- **Сортировка:** Автоматическое определение целевого склада. Крючок -> Рыболовный, Патрон -> Боевой.
- **Поиск:** Исключение проверки непрофильных складов при дефиците конкретного ресурса.
