# 2.1. ЛАГЕРЬ ТЕХ. ОБСЛУЖИВАНИЯ (THE WORKSHOP)

## 1. СТРАТЕГИЧЕСКОЕ НАЗНАЧЕНИЕ
Центральный узел реализации протоколов «Цех» и «К.У.Л.А.К.». Место глубокого обслуживания вооружения, оптики и снаряжения ТТсКО.

## 2. ИНФРАСТРУКТУРА И ЗОНИРОВАНИЕ
- **Объект:** Большая палатка (Large Tent) + Маскировочная сеть.
- **Очаг (Бочка/Печь):** Обязательный элемент. Используется для стерилизации, обжига и поддержания температурного режима при работе с маслами.
- **Рабочие поверхности:** Использование столов или пней. 
- **Разумение АН:** Категорический запрет на разборку механизмов ПСО-1 и СВД на открытом грунте.

## 3. ТЕХНИЧЕСКИЙ РЕСУРС (ЛОГИСТИЧЕСКИЙ ХВОСТ)
Концентрация специализированных наборов:
- Оружейное масло (Cleaning Kits) — расширенный запас.
- Точильные камни (Whetstones) — стационарные и переносные.
- Портняжный узел: Швейные наборы, кожаные иглы, запасы ткани ТТсКО.

## 4. РЕЖИМ «ШУМНОГО КРАФТА»
- **Условие:** Активация тех. работ только после 3-кратного цикла `LookAround360`.
- **Маскировка звука:** Приоритет работ во время дождя или сильного ветра для скрытия акустического следа заточки и чистки.
- **Мусорная гигиена:** Обрезки ткани и пустые наборы утилизируются в огне бочки для сокрытия следов пребывания "Мастера".
