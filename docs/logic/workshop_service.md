# 2.1. МАСТЕРСКАЯ КАК ТЕХНО-ХАБ (SERVICE LOGIC)

## 1. ПСИХОЛОГИЯ «ГОСТЕПРИИМСТВА ОХОТНИКА»
- **Визуальное доминирование:** Демонстрация быта (чистка картофеля, работа с ПСО-1, использование НЗ-1). Игрок должен видеть порядок ветерана.
- **Ограничение доступа:** Клиент допускается только к "внешнему" оборудованию (бочка, верстак-пень). Доступ к внутренним ячейкам большой палатки для чужаков заблокирован.

## 2. БЕЗОПАСНОСТЬ И КОНТРОЛЬ КЛИЕНТА
- **Зона ожидания:** Клиенту назначается точка пребывания (например, у костра/бочки). Любое движение за спину АН или к инвентарю — немедленное прерывание работ и нацеливание оружия.
- **Слепая навигация:** Вход и выход клиента из зоны Мастерской осуществляется через "петли" и густую растительность, чтобы минимизировать шанс запоминания координат.

## 3. ЭКОНОМИКА «УСЛУГ»
- **Аренда НЗ-1:** Возможность для клиента подточить нож или зашить одежду инструментами Некрасова за плату (патроны/еда).
- **Бартер трудом:** Система "Вывод за работу". Клиент помогает с рутинными задачами (полив в Agro-Hub, перенос дров) в обмен на безопасность пути.

## 4. ИНФРАСТРУКТУРА ХАБА
- **Центральный объект:** Умная бочка (вода + очаг).
- **Рабочая зона:** Верстак из пня/ящика для тонкого обслуживания оптики.
- **Информационная гигиена:** Основной "Золотой фонд" (валюта, ключи) перемещен в Deep Cache (1.3) перед приводом клиента.
