# РАЗДЕЛ 1: ОСОЗНАНИЕ ПУСТОТЫ (ZERO START LOGIC)

## 1. ПЕРВИЧНЫЙ ОПРОС ТЕЛА (BIOLOGY SCAN)
Сразу после инициализации АН выполняет проверку:
- **Health/Blood:** Оценка целостности.
- **Energy/Water:** Лимит времени до критического истощения.
- **Inventory:** Подтверждение отсутствия инструментов (Status: NULL).

## 2. АЛГОРИТМ «ГЛАЗА И ЗЕМЛЯ» (SEARCH MODE)
АН игнорирует зомби и здания на дистанции > 50м. Его фокус сужается до поверхности:
- **Поиск текстур:** Дороги, железнодорожные насыпи, береговая линия.
- **Цель:** Найти 2 малых камня (Stone_Small).
- **Разумение:** «Без камня я — жертва, с камнем — мастер».

## 3. ПЕРВЫЙ КРАФТ (THE FIRST EDGE)
Как только 2 камня в руках:
- **Действие:** Изготовление каменного ножа.
- **Трансформация:** Переход из статуса "Младенец" в статус "Вооружен".

## 4. ГИГИЕНА БЕРЕГА
- **LookAround360:** Выполняется после поднятия каждого камня.
- **Эвакуация:** Немедленный уход с открытого берега в "Зеленку" (лес) сразу после получения ножа.
